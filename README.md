# Student Management REST API

A beginner-level Spring Boot project that demonstrates how to build REST APIs using Java.

## What this project does
- Add a student
- View all students
- View a student by ID
- Delete a student.

The project uses in-memory storage (ArrayList) and does not use a database.

## Tech Stack
- Java
- Spring Boot
- REST APIs
- Maven
- Postman

## API Endpoints
| Method | Endpoint |
|------|----------|
| GET | /students |
| POST | /students |
| GET | /students/{id} |
| DELETE | /students/{id} |

## How to run
1. Clone the repository
2. Open in IntelliJ
3. Run `StudentapiApplication.java`
4. Server starts on port 8080
